
✔️ Новые MTProto-прокси с улучшенной маскировкой

Разработчики Telegram обновили работу MTProto, и теперь появилась возможность значительно лучше маскировать трафик.
В связи с этим мы запускаем вторую версию прокси с новым режимом работы.

➖➖➖➖➖➖➖➖➖➖

📦 Cascade Mode
┗ Это классическая схема с маршрутизацией через несколько серверов:

• подключение идёт к российскому серверу (вход)
• далее трафик уходит на немецкий сервер (выход)

Именно поэтому режим называется 📦 Cascade Mode

Такой прокси не маскируется под обычный трафик, как новый вариант, но в этом и нет необходимости:
для провайдера вы подключены к серверу в России, а фактически используете Telegram через Германию.

➖➖➖➖➖➖➖➖➖➖

📚 Obfuscated Mode
┗ Это новый формат MTProto-прокси.

❗️Требуется последняя версия Telegram на всех устройствах

Особенности:
• трафик маскируется под HTTPS (FakeTLS)
• используется дополнительное шифрование
• подключение идёт напрямую к немецкому серверу
• не требуется разделение на вход/выход

По сути это изолированное защищённое соединение для Telegram, которое сложнее определить и заблокировать (DPI/ТСПУ).

➖➖➖➖➖➖➖➖➖➖

ℹ️ Что выбрать?

Оба варианта работают по-разному, и “лучшего” нет:

у кого-то стабильнее работает 📦 Cascade Mode
у кого-то лучше показывает себя 📚 Obfuscated Mode

👉 Рекомендуем добавить оба варианта и использовать тот, который лучше работает у вас.
